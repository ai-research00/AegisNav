# Development Roadmap

## Phase 1: Core Offline Navigation
- Implement offline map loading
- Integrate GraphHopper or OSRM for routing
- Sensor fusion for compass and step tracking

## Phase 2: AR Guidance
- AR overlay of directional arrows and landmarks
- Anchor POIs to map landmarks
- Fallback 2D map mode

## Phase 3: Voice and UI Enhancements
- Offline text-to-speech guidance
- Multi-language support
- Visual icons and distance indicators

## Phase 4: Testing and Calibration
- Field tests in GPS-denied environments
- Sensor calibration routines
- AR alignment accuracy tests

## Phase 5: Security & Resilience
- Encrypt map tiles and route data
- Ensure offline operation is robust
- Implement alerts for sensor drift
