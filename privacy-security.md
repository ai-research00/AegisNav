# Privacy and Security Guidelines

## Offline Operation
- All data remains on device
- No internet required for core functionality
- No cloud logging of user location

## Encryption
- Map tiles encrypted using device's keystore
- Route and waypoint data stored securely
- Optional password protection for map regions

## Minimal Permissions
- Sensors: accelerometer, gyroscope, magnetometer
- Storage: local map and route data
- Camera: AR mode only
- No network permissions needed for navigation

## Resilience
- Detect compass drift or sensor error
- Prompt user to recalibrate
- Provide offline fallback maps
