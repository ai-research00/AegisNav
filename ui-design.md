# User Interface (UI) Design Guidelines

## AR Compass View
- Full-screen camera feed
- Arrow pointing to next waypoint
- Distance to waypoint displayed
- Landmark labels anchored to real-world features

## Map View (Fallback)
- Top-down map
- Current location marker
- Route highlighted
- Zoom/pan offline

## Visual Style
- Minimalist, high contrast for low-light
- Intuitive icons for turns, landmarks, and obstacles
- Optional dark/light theme
